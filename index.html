Currículo

O que vais aprender

⌨️

Programação

Java, Python, HTML/CSS/JS e metodologias modernas de desenvolvimento de software.

  • Programação orientada a objetos
  • Desenvolvimento web full-stack
  • Algoritmos e estruturas de dados
🗄️

Bases de Dados

Modelação relacional, SQL avançado e introdução a bases de dados NoSQL.

  • Modelação entidade-relação
  • SQL e stored procedures
  • Administração de SGBD
🖧

Sistemas & Redes

Arquitetura de sistemas, redes TCP/IP, administração Linux e Windows Server.

  • Configuração de redes
  • Segurança informática
  • Virtualização e cloud
📐

Engenharia de Software

Análise de requisitos, UML, padrões de design e gestão de projetos ágeis.

  • Scrum e metodologias ágeis
  • Modelação UML
  • Testes e qualidade de software
